# Buscador de Auditorías por Requerimiento

Esta aplicación permite buscar auditorías en función de requerimientos usando lenguaje natural en español.

## Requisitos

- Tener Python 3 instalado en tu PC.
- Instalar las dependencias necesarias ejecutando el siguiente comando en la terminal:

  pip install streamlit pandas scikit-learn openpyxl

## Archivos incluidos

- auditoria_app.py → Código de la aplicación Streamlit.
- Requierimientos de Auditoria.xlsx → Archivo con los datos de auditorías y requerimientos.
- README.txt → Este archivo con instrucciones.

## Cómo ejecutar la aplicación

1. Abrí una terminal o consola de comandos en la carpeta donde están estos archivos.
2. Ejecutá el siguiente comando:

   streamlit run auditoria_app.py

3. Se abrirá automáticamente una ventana en tu navegador con la interfaz de búsqueda.

## Cómo usar

- Ingresá una consulta en lenguaje natural, por ejemplo:

  ¿Dónde se pidió evidencia de capacitación?

- La aplicación mostrará la auditoría más relevante y el requerimiento asociado.
